#### 介绍
2021年7月7日上线的浩瀚银河快视频Web可信解决方案。

#### 软件架构
采用PHP+H5+CSS3。前端主流架构。

#### 使用教程

1.  网站部署PHP、Nginx，PHP不低于7.0。
2.  丢进网站下文件夹，自行设置Nginx高并发参数。
3.  正常访问即可。
